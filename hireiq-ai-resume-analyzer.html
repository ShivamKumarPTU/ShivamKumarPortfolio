HireIQ — AI Resume Analyzer

A full-stack web application that analyzes resumes using Google Gemini AI — extracting skills, evaluating quality, suggesting improvements, and recommending matching jobs via the Adzuna API.

View Live App View on GitHub

What HireIQ Does

🧠

AI Resume Analysis

Uses Google Gemini AI to deeply analyze uploaded resumes, identifying strengths, weaknesses, and areas for improvement.

🎯

Skill Extraction

Automatically identifies and categorizes technical and soft skills from any resume format.

💡

Improvement Suggestions

Provides actionable, specific suggestions to improve resume score and increase interview chances.

🔍

Job Matching

Integrates Adzuna API to suggest real job openings that match the candidate's skill profile.

🔐

Secure Auth

Email verification and password reset flows using Brevo for secure, production-ready authentication.

📊

Resume Scoring

Generates a quantitative resume score with detailed breakdown so candidates know exactly where they stand.

Tech Stack

React.js Spring Boot Google Gemini AI MySQL Adzuna API Brevo (Email) Java REST API
